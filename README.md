# Brand Guidelines - Pipiles

Esta carpeta contiene solo los archivos necesarios para publicar el manual de marca de Pipiles.

## Para subir a GitHub Pages

Sube el contenido completo de esta carpeta:

- `index.html`
- `assets/`
- `.nojekyll`

No necesitas subir los otros archivos de la carpeta principal de Pipiles.

## Archivo principal

GitHub Pages debe abrir:

```text
index.html
```

## Estructura

```text
Brand Guidelines/
  index.html
  .nojekyll
  assets/
    css/
    images/
    js/
    logos/
```
